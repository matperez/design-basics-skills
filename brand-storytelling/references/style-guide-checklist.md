# Style guide — чеклист компонентов

## Структура style guide

Рекомендуемый объём: 25–35 страниц. Ссылки на дополнительные ресурсы — inline.

### 1. Brand story

- [ ] Миссия (1–2 предложения: какую проблему решаем)
- [ ] Видение (куда стремимся)
- [ ] Ценности (3–5 слов с расшифровкой)
- [ ] Обещание бренда (что получает клиент)

### 2. Целевая аудитория

- [ ] Персоны (1–3 ключевых)
- [ ] Демография (возраст, гео, контекст)
- [ ] Потребности и мотивации
- [ ] Болевые точки

### 3. Визуальная идентичность

#### Логотип
- [ ] Основная версия
- [ ] Допустимые вариации (горизонтальная, вертикальная, иконка)
- [ ] Монохромная версия (ч/б, white-on-dark)
- [ ] Clear space (минимальные отступы)
- [ ] Минимальный размер
- [ ] Запретные варианты (stretch, recolor, shadow, overlay)

#### Цвета
- [ ] Primary palette (1–2 цвета) с кодами: HEX, RGB, CMYK, Pantone
- [ ] Secondary palette (2–3 цвета)
- [ ] Neutral palette (grays, white, black)
- [ ] Semantic colors (success, error, warning, info)
- [ ] Правила использования: что dominant, что accent
- [ ] Примеры accessible пар (контраст WCAG AA)

#### Типографика
- [ ] Семейства шрифтов (headline, body, accent)
- [ ] Размеры (type scale)
- [ ] Weights для каждой роли
- [ ] Line-height, letter-spacing
- [ ] Fallback fonts (для web, email)

#### Иконографика и иллюстрации
- [ ] Стиль иконок (outline, filled, rounded)
- [ ] Размер grid
- [ ] Стиль иллюстраций (flat, isometric, hand-drawn)
- [ ] Стиль фотографий (натуральные, постановочные, фильтры)

### 4. Голос бренда

- [ ] 3–5 прилагательных тона (например: modern, clever, caring)
- [ ] Вариации по каналам:
  - Соцсети (casual, emoji допустимы / нет)
  - Сайт (профессиональный, но не сухой)
  - Email (дружелюбный, помогающий)
  - Поддержка (эмпатичный, конкретный)
- [ ] Примеры «как мы звучим» vs «как мы НЕ звучим»

### 5. Гайдлайны по тексту

- [ ] Уровень читаемости (grade level / целевая сложность)
- [ ] Грамматические правила (Oxford comma, числа словами vs цифрами)
- [ ] Бренд-слова (всегда используем) и запретные слова
- [ ] Форматирование: заголовки, списки, CTA
- [ ] Capitalization rules

## Процесс создания

| Шаг | Действие | Кто участвует |
|-----|----------|---------------|
| 1 | **Outline** — оглавление, структура | Бренд-команда |
| 2 | **Наполнение** — визуал + копирайтинг + accessibility | Дизайнеры + копирайтеры |
| 3 | **Ревью** — стейкхолдеры внутри и вне команды | Creative director + leadership |
| 4 | **Публикация** — доступ для всех | Вся компания + фрилансеры |
| 5 | **Обновление** — каждые 6–12 месяцев | Бренд-команда |

## 3 правила эффективного style guide

1. **Сделай доступным** — опубликуй где все найдут; добавь интерактивность (вопросы, запросы).
2. **Обновляй регулярно** — бренд живой; style guide тоже.
3. **Показывай dos and don'ts** — реальные примеры «как надо» и «как не надо» в разных контекстах.
