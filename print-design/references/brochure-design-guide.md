# Дизайн брошюры: сгибы, панели и контент-стратегия

## Зачем брошюра

Брошюра — физический носитель, который можно подержать, полистать, оставить на столе. В мире digital-рекламы это преимущество: тактильный контакт повышает запоминаемость. Брошюра подходит, когда нужно рассказать больше, чем помещается на флаер, но компактнее, чем каталог.

## 3 главных типа сгибов (с раскладкой панелей)

### Bi-fold (пополам)

```
Внешняя сторона:          Внутренняя сторона:
┌──────────┬──────────┐   ┌──────────┬──────────┐
│          │          │   │          │          │
│  Задник  │ Обложка  │   │  Внутр.  │  Внутр.  │
│          │          │   │  левая   │  правая  │
│          │          │   │          │          │
└──────────┴──────────┘   └──────────┴──────────┘
```

**4 панели.** Самый простой формат.

| Панель | Контент |
|--------|---------|
| Обложка | Заголовок + hero-визуал + лого |
| Внутренняя левая | Intro, mission, «о нас» |
| Внутренняя правая | Features, услуги, pricing |
| Задник | Контакты, карта, QR, соцсети |

### Tri-fold (C-fold)

```
Внешняя сторона:                    Внутренняя сторона:
┌──────────┬──────────┬─────────┐   ┌─────────┬──────────┬──────────┐
│          │          │         │   │         │          │          │
│  Задник  │  Сред-   │ Обложка │   │ Внутр.  │  Внутр.  │  Внутр.  │
│          │  няя     │         │   │ правая  │  центр   │  левая   │
│          │          │         │   │ (флап)  │          │          │
└──────────┴──────────┴─────────┘   └─────────┴──────────┴──────────┘
```

**6 панелей.** Самый популярный формат для маркетинга.

| Панель | Контент | Заметки |
|--------|---------|---------|
| Обложка | Заголовок + визуал + CTA «открой» | Первое впечатление = решает всё |
| Внутренняя правая (флап) | Intro, обзор, тизер | Первая панель при открытии; чуть уже! |
| Внутренний центр | Основной контент: features, шаги | Самая большая видимая площадь |
| Внутренняя левая | Детали, testimonials, FAQ | Продолжение narrative |
| Средняя внешняя | Дополнительная info, QR | Видна при частичном открытии |
| Задник | Контакты, карта, лого, соцсети | Часто упускается — а зря! |

**Важно:** внутренняя правая панель (флап) должна быть **на 2–3mm уже** — иначе выступает.

### Gatefold

```
Внешняя сторона:                    Внутренняя сторона:
┌────────┬────────────┬────────┐   ┌────────┬────────────┬────────┐
│        │            │        │   │        │            │        │
│ Левая  │   Центр    │ Правая │   │ Левая  │   Центр    │ Правая │
│ створка│  (видна    │створка │   │ внутр. │  (reveal)  │ внутр. │
│        │  снаружи)  │        │   │        │            │        │
└────────┴────────────┴────────┘   └────────┴────────────┴────────┘
         ←створки закрываются→
```

**6 панелей.** Эффект «открытия ворот» = wow-момент.

| Панель | Контент |
|--------|---------|
| Внешний центр (видна при закрытых створках) | Тизер, интрига, минимум текста |
| Створки (снаружи) | Визуал или продолжение тизера |
| Внутренний reveal (при открытии) | Главный визуал / hero product / ключевое сообщение |
| Створки (внутри) | Поддержка: features, цифры, отзывы |

## Контент-стратегия для брошюры

### Принцип «одна тема — одна панель»

Каждая панель решает одну задачу. Не смешивай pricing и testimonials на одной панели.

### Flow (порядок чтения)

1. **Обложка** — зацепи. Заголовок + визуал + CTA.
2. **Первая внутренняя** — объясни. Кто мы / что предлагаем.
3. **Основной разворот** — раскрой. Features, шаги, выгоды.
4. **Финал** — убеди. Testimonials, цифры, гарантии.
5. **Задник** — действуй. Контакты, QR, CTA.

### Copy для брошюры

| Правило | Почему |
|---------|--------|
| Lead with benefit, not feature | «Экономьте 2 часа в день» > «Автоматизация процессов» |
| Заголовки — сканируемые | Читатель листает, не читает подряд |
| Bullet points > параграфы | Проще сканировать |
| Один CTA на брошюру | Не размывай внимание |
| Реальные testimonials | С именем и фото — доверие выше |

### Визуал для брошюры

- **Единая сетка** для всех панелей — consistency.
- **Повторяющиеся элементы:** footer с лого, номера страниц, линии-разделители.
- **Изображения ≥ 300 DPI** — обязательно для печати.
- **Charts / infographics** — для complex data (лучше, чем текст).
- **White space** — не заполняй каждый мм; дай контенту дышать.

## Бумага и финиш

| Тип бумаги | Ощущение | Когда |
|-----------|----------|-------|
| **Gloss** | Яркий, блестящий | Фотографии, product showcases |
| **Matte** | Современный, тактильный | Текстовые брошюры, premium feel |
| **Uncoated** | Натуральный, крафтовый | Eco-бренды, handmade, community |
| **Silk / Satin** | Между gloss и matte | Универсальный баланс |

**Плотность:** 170–250 г/м² для брошюр; 300+ г/м² для обложек и открыток.

## Чеклист перед печатью брошюры

- [ ] Все панели правильного размера (флап уже на 2–3mm для tri-fold)
- [ ] Сгибы отмечены на макете
- [ ] Контент не попадает на линию сгиба (или сделано осознанно)
- [ ] CMYK, 300 DPI, bleeds 3mm
- [ ] Шрифты embedded / outlined
- [ ] Пробная печать + тестовый сгиб
- [ ] Вычитка текста на каждой панели
- [ ] QR-код тестирован (сканируется?)
