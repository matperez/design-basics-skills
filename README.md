# Design basics — Agent Skills

Набор из семи самодостаточных скиллов для агентов в Cursor: основы UI/UX, веб-дизайна, типографики, цвета, печати, бренда и прототипирования. Скиллы не привязаны к конкретному инструменту дизайна.

Каждый скилл — не справочник, а **пошаговый процесс** (дерево решений), который агент проходит вместе с пользователем: от брифа / формулировки задачи до готового результата.

## Структура репозитория

Каждый скилл — отдельная папка:

- `SKILL.md` — YAML frontmatter (`name`, `description`) и основной текст с шагами процесса, матрицами выбора и чеклистами.
- `references/` — развёрнутые справочники, каталоги и чеклисты; пути относительные к папке скилла.

## Скиллы

| Папка | `name` | Процесс | Когда полезен |
|-------|--------|---------|----------------|
| `ui-ux-principles` | `ui-ux-principles` | От проблемы до валидированного интерфейса (8 шагов): бриф → исследование → IA → восприятие (Gestalt, Fitts, иерархия) → UI-принципы → дизайн-система → accessibility → итерации | UX/UI стратегия, исследования, доступность, дизайн-системы, визуальное восприятие, этика |
| `prototype-wireframe` | `prototype-wireframe` | От идеи до валидированного прототипа (7 шагов): гипотеза → user flow → fidelity → wireframe → прототип → MVP scope → полный цикл приложения | Wireframe, mockup, прототипы, user flow, MVP, app design |
| `web-design` | `web-design` | От стратегии до запуска сайта (8 шагов): стратегия → структура → layout/grid → навигация/CTA → визуал → responsive → лендинги → запуск | Сайты, responsive, mobile-first, лендинги, CTA, сетки, редизайн |
| `typography` | `typography` | Выбор шрифтов под задачу (6 шагов): контекст → категория → конкретный шрифт → пара → иерархия → технические настройки | Подбор шрифтов, пары, иерархия, кернинг, интерлиньяж, web и бренд |
| `color-theory` | `color-theory` | Построение палитры (7 шагов): настроение → base hue → схема гармонии → 60-30-10 → accessibility → RGB/CMYK → тестирование | Палитры, контраст, accessibility, цвет для UI и бренда |
| `print-design` | `print-design` | От брифа до press-ready файла (7 шагов): цель → размер/сгиб → материалы/copy → layout → типографика/цвет → pre-press → открытки | Флаеры, брошюры, открытки, подготовка к печати, CMYK |
| `brand-storytelling` | `brand-storytelling` | Построение и коммуникация бренда (6 шагов): идентичность → логотип → style guide → продуктовый сторителлинг → презентации → резюме | Логотипы, style guide, бренд-нарратив, презентации, pitch decks |

В `description` у скиллов указаны перекрёстные ссылки на другие скиллы из этого набора (например `color-theory` и `typography` используются внутри `brand-storytelling` и `web-design`).
