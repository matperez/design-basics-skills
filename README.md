# Design basics — Agent Skills

Набор из семи самодостаточных скиллов для агентов в Cursor: основы UI/UX, веб-дизайна, типографики, цвета, печати, бренда и прототипирования. Скиллы не привязаны к конкретному инструменту дизайна.

## Структура репозитория

Каждый скилл — отдельная папка:

- `SKILL.md` — YAML frontmatter (`name`, `description`) и основной текст.
- `references/` (если есть) — развёрнутые заметки и иллюстрации; пути относительные к папке скилла.

## Скиллы

| Папка | `name` | Когда полезен |
|-------|--------|----------------|
| `ui-ux-principles` | `ui-ux-principles` | UX/UI, исследования, доступность, дизайн-системы, Gestalt, закон Фитца, иерархия — не вместо веб-раскладки и не вместо прототипирования. |
| `prototype-wireframe` | `prototype-wireframe` | Wireframe, mockup, прототипы, user flow, MVP, проверка идей до кода. |
| `web-design` | `web-design` | Сайты: контент, макет, навигация, responsive, mobile-first, лендинги, CTA, сетки. |
| `typography` | `typography` | Шрифты, пары, иерархия, кернинг, интерлиньяж, трекинг, экран и бренд. |
| `color-theory` | `color-theory` | Круг, RYB/RGB/CMYK, схемы гармонии, контраст, палитры, UI и бренд. |
| `print-design` | `print-design` | Флаеры, брошюры, открытки, размеры, иерархия, CTA, подготовка к печати. |
| `brand-storytelling` | `brand-storytelling` | Сторителлинг, типы логотипов, презентации, style guide, резюме, воркшопы. |

В `description` у части скиллов указаны перекрёстные ссылки на другие скиллы из этого набора (например `web-design`, `prototype-wireframe`).
