# Color theory

**Теория цвета** для UI, бренда и печати: круг, гармонии, RGB/CMYK, контраст.

## Зачем теория цвета в продукте

- **Визуальный эффект** — контраст для акцентов, баланс для спокойных интерфейсов.
- **UX** — гармония помогает сканировать, выделять действия, читать состояния.
- **Бренд** — палитра передаёт характер и настроение.

## База: три типа на цветовом круге (модель RYB)

1. **Primary** — красный, жёлтый, синий как опоры смешения.
2. **Secondary** — смеси двух основных (оранжевый, зелёный, фиолетовый).
3. **Tertiary** — смесь первичного и вторичного (красно-оранжевый и т.д.).

Для **экрана** — **RGB** (аддитивная модель, 16 млн+ цветов).
Для **печати** — **CMYK** (субтрактивная модель, 16 000+ комбинаций).

> Подробнее о RGB/CMYK с иллюстрациями: `references/rgb-cmyk-guide.md`.

## 7 схем гармонии

1. **Analogous** — соседи на круге; один главный, второй поддерживающий, третий акцент.
2. **Monochromatic** — оттенки одного цвета; узнаваемые монобренды.
3. **Complementary** — противоположности на круге; сильный контраст.
4. **Split-complementary** — база + два соседа от комплемента; богаче палитра.
5. **Triadic** — три равноудалённых; один доминирует.
6. **Tetradic** — два пары комплементов; максимальный контраст, нужна дисциплина.
7. **Square** — четыре равноудалённых; смелые, «игровые» палитры.

> Подробнее с примерами палитр: `references/color-palettes-guide.md`.

## Hue, Value, Saturation

- **Hue** — чистый цвет (красный, синий, зелёный).
- **Value** — количество затемнения (shade = + чёрный, tint = + белый).
- **Saturation** — интенсивность цвета (vivid → dull).

Насыщенный красный = предупреждение. Приглушённый = спокойная нотка. Светлый на светлом = нечитаемо.

## Психология цветовой температуры

- **Тёплые** (красный, оранжевый, жёлтый) — энергия, страсть, срочность. Coca-Cola, Red Bull.
- **Холодные** (синий, зелёный, фиолетовый) — доверие, спокойствие, профессионализм. IBM, Blue Shield.
- **Нейтральные** (чёрный, белый, серый) — фон, акцент, баланс.

Символика цвета **контекстуальна по культуре** — не абсолютна.

## Практика для агента

- Разделяйте **семантику UI** (успех, ошибка, ссылка, disabled) и **брендовые** цвета.
- Проверяйте **контраст текста и фона** (WCAG 4.5:1 для обычного текста, 3:1 для крупного).
- Не смешивайте на одном экране много равноправных насыщенных цветов без нейтрали.
- **Символика и психология** — используйте как ориентир, но не как абсолют.
