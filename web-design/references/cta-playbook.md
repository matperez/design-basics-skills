# CTA Playbook — Call-to-Action для веб-дизайна

## Что такое CTA

CTA (Call-to-Action) — элемент интерфейса, побуждающий пользователя к действию: покупке, регистрации, скачиванию, подписке. Персонализированные CTA увеличивают конверсию до 202%.

## Форматы CTA

| Формат | Описание | Когда |
|--------|---------|-------|
| **Кнопка** | Чёткий clickable элемент | Основной CTA на странице |
| **Текстовая ссылка** | Встроена в контент | Вторичные действия, inline |
| **Изображение** | Кликабельный визуал с overlay | E-commerce, баннеры |
| **Pop-up / slide-in** | Появляется при определённом триггере | Подписка, специальные предложения |
| **Форма** | Поля ввода + кнопка submit | Lead generation, sign-up |
| **Баннер** | Фиксированная полоса (top/bottom) | Промо, announcements |
| **Video CTA** | Промпт внутри видео | Video landing pages |

## 7 правил эффективного CTA

### 1. Одно главное действие на экран

Каждый экран (viewport) — один primary CTA. Несколько конкурирующих CTA = ни одного.

### 2. Язык = ценность, не инструкция

| Плохо | Хорошо |
|-------|--------|
| «Click here» | «Start free trial» |
| «Submit» | «Get your report» |
| «Learn more» (без контекста) | «See how it works» |

Начинай с глагола. Покажи, что пользователь **получает**.

### 3. Визуальный контраст

- CTA — самый контрастный элемент на экране.
- Используй brand accent color.
- White space вокруг CTA усиливает внимание.
- Размер: достаточно большой для уверенного клика/тапа (≥ 44×44px).

### 4. Стратегическое размещение

| Позиция | Когда |
|---------|-------|
| **Above the fold** | Всегда — первый CTA виден без скролла |
| **After value proposition** | После объяснения ценности |
| **After social proof** | После отзывов / логотипов клиентов |
| **Bottom of page** | Финальный CTA для тех, кто дочитал |
| **Sticky (floating)** | Для длинных страниц — CTA следует за скроллом |

### 5. Progression — CTA усложняется

По мере скролла пользователь узнаёт больше → CTA становится конкретнее:
- Верх: «Learn more»
- Середина: «Start free trial»
- Низ: «Buy now — 20% off»

Пример: Jira — от «Get started» к «Get started with Jira Product Discovery».

### 6. Urgency без манипуляции

| Этично | Неэтично |
|--------|----------|
| «Free trial — no credit card» | «Only 2 left!» (если неправда) |
| «Offer ends Sunday» (реально) | Fake countdown timer |
| «Join 10,000+ teams» | «You're missing out!» (FOMO abuse) |

### 7. A/B тестируй

- Текст кнопки (verb, value, length)
- Цвет кнопки
- Размер и padding
- Placement
- Формулировку value proposition рядом

## Примеры CTA по индустриям

| Бренд | CTA | Почему работает |
|-------|-----|----------------|
| **Figma** | «Get started for free» | Бесплатность убирает барьер |
| **Netflix** | «Get Started» + sticky при скролле | Множественные touchpoints |
| **HelloFresh** | «Get Offer» | Scarcity + value |
| **Headspace** | «Try it for free» → «Get your headspace» | Free trial + emotional branding |
| **Philips** | «Shop new» / «Shop best-sellers» | Specific > generic |
| **Slack** | «Try for free» + «Talk to sales» | Два пути: self-serve + enterprise |
| **Notion** | «Get Notion free» | Бесплатность + бренд в CTA |

## CTA чеклист

- [ ] Primary CTA виден above the fold.
- [ ] Текст — глагол + ценность.
- [ ] Контраст достаточный (визуально выделяется).
- [ ] Touch target ≥ 44×44px.
- [ ] CTA повторяется 2–3 раза на длинных страницах.
- [ ] Рядом с CTA — supporting copy (почему кликнуть).
- [ ] Нет конкурирующих CTA одинаковой важности.
- [ ] A/B тест проведён или запланирован.
