# Typography

Правила и практика **типографики** для экрана и бренда.

## Зачем типографика

- **Привлечь внимание** до чтения.
- **Иерархия** — размеры, веса, пары шрифтов, контраст ролей (заголовок / подзаголовок / body).
- **Узнаваемость бренда** — стабильные решения по семействам и стилям.
- **Поддержка смысла** — читабельность (legibility/readability) и тон.

## Пять классов шрифтов

| Класс | Характер | Типичное применение |
|--------|-----------|---------------------|
| **Serif** | традиционный, «редакционный» | бренды, длинные тексты (с оговорками по экрану) |
| **Sans serif** | современный, нейтральный | UI, стартапы, экранный body |
| **Script** | рукописный, выразительный | короткие акценты, не длинный набор |
| **Monospace** | фиксированная ширина | код, ретро-эстетика |
| **Display** | декоративный | логотипы, постеры, крупные заголовки |

**Терминология:** typeface (семейство) vs font (начертание внутри семейства, напр. Regular / Bold).

## Ключевые параметры

- **Alignment** — единые правила для блоков; длинные тексты чаще по левому краю; короткие заголовки — по центру при необходимости.
- **Цвет текста** — контраст к фону; ориентиры доступности (WCAG); насыщенность для заголовков vs нейтраль для абзацев.
- **Hierarchy** — H1 > H2 > body; белое пространство и вес усиливают порядок, не только кегль.
- **Kerning** — интервал между **конкретными** парами букв.
- **Leading (line-height)** — часто ~1.125–1.2 от кегля как стартовая зона; подгонять под выбранный шрифт.
- **Tracking** — общий разряд по строке/слову; капс иногда читается лучше с чуть большим трекингом.

## Как подбирать шрифт (процесс для агента)

1. Контекст носителя (экран / печать) и **длина текста**.
2. Характер бренда (строгий, игровой, премиальный).
3. Пара для заголовок + body: **контраст ролей**, не два конкурирующих декоративных.
4. Проверка **читаемости** в малых размерах и на мобильном.
5. Лицензии и веб-форматы — по запросу пользователя.
